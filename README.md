# ⚛️ React Series from Scratch

A comprehensive React.js learning repository that covers React fundamentals to advanced concepts through hands-on examples and projects. This repository is intended for beginners as well as developers who want to strengthen their React knowledge.

## 📚 Topics Covered

- Introduction to React
- JSX
- Components
- Props
- State
- Event Handling
- Conditional Rendering
- Lists & Keys
- Forms
- Hooks
  - useState
  - useEffect
  - useContext
  - useReducer
  - useMemo
  - useCallback
  - useRef
- React Router
- Context API
- API Integration
- Custom Hooks
- Performance Optimization
- Error Handling
- React Project Structure
- Best Practices
- Mini Projects
- Deployment

## 🛠️ Tech Stack

- React.js

- HTML5

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/React-Series-from-Scratch.git
```

### Navigate to the project

```bash
cd React-Series-from-Scratch
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
